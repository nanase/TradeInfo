TradeInfo

TradeInfo は村人の取引状況を抽出し、閲覧する Bukkit プラグインです。

利用方法

プラグインを導入し、一定間隔おきにプレイヤーの周辺の村を探索します (初期値では 5 分)。村が存在すると、村の情報を抽出します。更新されるのは、近くに存在する村の情報のみです。
閲覧画面では村人を 1行として取引の内容を表示します。各セルでは 売りアイテム、買いアイテム、取引回数 が表示されています。セルが赤くなったものは最大取引回数に達してそれ以上取引ができないことを表しています。
アイテムは数量が数字で表示され、さらにエンチャントについては「E」と表示されます。エンチャントの内容についてはアイテムアイコンをポイントすることで詳細を表示できます。

用語について

座標
村の中心座標です。それぞれ (X, Y, Z) を表します。この座標は村の存在判定に用いられるもので、変動する場合があります。
距離
スポーン地点から村の中心座標までの距離です。
半径
村の大きさを表す半径です。中心座標と半径を用いて村人の判定をしています。
家屋
村に存在する家屋 (ドアの数) です。ある条件を満たした場合のみドアであると判定されます。状況により、家屋数は大きく変動します。また、実際のドアの数と一致するとは限りません。
推定人口
村の推定の人口です。「推定」とは、繁殖時に用いられる村人のおおよその人口を意味し、実際に存在する村人の人口とは異なります。
繁殖
家屋と推定人口による繁殖の可否を表します。村ではドアの数の 35% と 大人の村人 の数が同数になるまで繁殖が継続します。
取引可能
取引可能な村人の人口を意味します。取引可能な村人とは以下の条件を全て満たした村人を意味します。
  • 大人である
  • 村の範囲内に存在している
  • 一回でも取引ダイアログを表示された
職業
村人の職業です。村人は 農夫、司書、司祭、鍛冶屋、肉屋 の何れかの職業に就きます。職業によって取引できる内容は異なります。
取引回数
村人と同じ商品を取引できる回数です。ゲーム中では表示されません。右向きの矢印の下に表示され、取引回数 / 最大取引回数 の形式で表示されます。取引は最大回数に達することでそれ以上の取引ができなくなります。ただし、ある条件を満たすと最大回数に達している取引に限り、最大回数が増加し、再び取引可能となります。

補足

  • このプラグインではプレイヤーがログインしていない場合は情報の収集を行いません。
  • このプラグインでは村の情報を改変する処理は行なっていません。また、表示内容については不安定な場合があります。
